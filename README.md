# dockereval
A simple docker tool to evaluate LLM

## Usage
### clone the repo
```bash
git clone --recursive https://github.com/adlsdztony/dockereval.git
cd dockereval
```
### Download the LLM model
download .gguf file

### edit .env
set the path to the LLM model in the .env file

### run with docker-compose
```bash
docker-compose up
```
